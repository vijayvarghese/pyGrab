# pyGrab
pyGrab is a simple web scraper which scraps images form https://pngtree.com for given keyword input. 

Usage : <br>
$ git clone https://github.com/vijayvarghese/pyGrab.git <br>
$ cd pyGrab<br>
$ pip install -r requirements.txt<br>
$ python pyGrab.py<br>
<br>


 <br>
Requirements <br>
* urllib<br>
* bs4<br>
* requests<br>
